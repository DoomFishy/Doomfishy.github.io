Memory Management Unit (MMU) Simulator

C | Page Tables | LRU | TLB | Demand Paging

Designed and implemented an MMU Simulator in C that reads virtual addresses from a file, translates them to physical addresses using a TLB and Page Table

Key features implemented

  • Demand paging
  • LRU page replacement when memory is full
  • Page fault handling
  • Measuring TLB hit rate and page faults rate

CPU Scheduling Simulator

C | FCFS | SJF | Round Robin | Priority | Hybrid Scheduling

Implemented 5 CPU scheduling algorithms.

Algorithms Implemented

  • First Come First Serve (FCFS)
  • Shortest Job First (SJF)
  • Round Robin (RR)
  • Priority
  • Hybrid: Priority and Round Robin for same priority tasks

Calculates average waiting time, turnaround time, and response time for each algorithm
